# loghell
Pretty simple log management tool

### [dashboard](./dashboard/README.md)
### [example app](./example/README.md)

### Usage
```bash
Usage of loghell
  -tcp int
    	set tcp server port (default 3031)
  -v	verbose output
  -ws int
    	set ws server port (default 3032)
    	
# start loghell
$ go run ./src/main.go
```
