# loghell
Pretty simple log management tool

### [example app](./example/README.md)

### Usage
```bash
Usage of loghell:
  -http int
    	set http server port (default 3033)
  -tcp int
    	set tcp server port (default 3031)
  -v	verbose output
  -ws int
    	set ws server port (default 3032)
    	
# start loghell
$ go run ./src/main.go
```
